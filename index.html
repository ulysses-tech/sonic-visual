소닉 비주얼 — 웨이브폼 원판 레코더

목소리를 녹음해 진폭을 분석하고, 목걸이·팔찌 제작용 원판 지름 목록을 자동 계산해 CSV로 내려받습니다. 서버 전송 없이 브라우저 안에서만 처리되며, 녹음 데이터는 저장되지 않습니다.

1. 녹음

마이크 권한을 허용한 뒤 녹음 시작을 눌러주세요
인식된 음성 텍스트

녹음하면 이 자리에 말한 내용이 표시됩니다

이 브라우저는 음성 인식(Web Speech API)을 지원하지 않아 텍스트 변환은 표시되지 않습니다. 파형 분석·CSV 기능은 정상 작동합니다.

2. 원판 설계 조건

12개
예: 2로 설정 시 지름이 2mm 단위(5, 7, 9…)로만 계산됩니다

3. 원판 미리보기

녹음을 마치면 원판 순서와 크기가 여기 표시됩니다

왼쪽부터 녹음 순서대로 배열됩니다. 실제 제작 시 이 순서 그대로 끈에 꿰어 조립하세요.

4. 데이터 내보내기
